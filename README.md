# Business Manager

This is the application for managing businesses on the wealthmarket. you can create or manage your existing businesses here. you must be registered to login and access this app.

## Instructions


---

### Setup Guidelines
1. clone the repository using the code below

    >```git clone https://github.com/The-Nigeria-Development-Foundation/BusinessManager.git```
1. create a branch for your module with the `"feature-"` prefix e.g `"feature-business-module"` on your system and on the repository online
1. checkout to your newly created branch


---

### Usage Guidelines
1. Run `ng serve` to run the app on a dev server. Navigate to `http://localhost:4200/` to view the app. The app will automatically reload if you change any of the source files.
